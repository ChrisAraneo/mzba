# mzba

# Server

## Prerequisites

- Java 8 or higher - download from https://www.oracle.com/java/technologies/downloads/
- Scala - download from https://www.scala-lang.org/
- sbt (Scala Build Tool) - will be installed with Scala

## Running the server

Navigate to the project directory:

```bash
cd mzba-game-server
```

### Compile with sbt

```bash
sbt compile
```

### Run with sbt

```bash
sbt run
```

# Development

### Run tests

```bash
sbt test
```
